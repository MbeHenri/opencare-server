# opencare-server

## Prerequisites

- Node.js 16+
- Yarn or NPM

## Installation

- Install dependencies

```bash
yarn install
```

- Create application configuration

```bash
cp .env.example .env
nano .env
```

- Start Application

```bash
yarn start
```

The application will be launched by [Nodemon](https://nodemon.com). It will restart automatically on file change
